== Description ==

The sensor data that we found in the office needs a place to make hisself useful, so we created a plugin. Fill in the url to the JSON-file, add and edit sensors and show some awesome data on the web-thingy.

Enjoy!

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/ignation-iot` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Ignation IOT screen to configure the plugin
4. Go to the ‘settings’-tab and fill in the url to the JSON file
5. Add some sensors
6. Add the plugin widget to a page and edit the style
7. Show some awesome data
